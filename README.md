# 🚄 세상을 잇고 채우는 트레:in(人)

AI 기반 철도 여행 플랫폼

## 📌 프로젝트 개요

철도 이동을 단순한 ‘이동 수단’이 아닌 **누구나 이용 가능한 맞춤형 여행 경험**으로 확장하는 AI 통합 서비스입니다.
열차 예매 시 AI 기반 시스템으로부터 **편의성 높은 최적 여행 경로, 좌석 조합, 관광 정보, AI 기반 맞춤 안내** 등을 제공받을 수 있습니다.

또한 지역 소상공인 정보, 관광 데이터, 보험 정보 등을 활용해
사용자에게는 다양한 여행 편의를, 지역에는 지속 가능한 **경제 및 사회적 활성화 효과**를 제공합니다.

📍 핵심기술

- 인공지능(AI)
- 웹/모바일 앱 개발
- 빅데이터
- NFC 등

## 🎯 프로젝트 동기

기존 대중교통 여행에서 느낀 **경로 파악의 어려움, 정보 분산, 외국인/교통약자의 이동 불편함** 등의 문제를 해결하고자 시작되었습니다.

여행이 **지역 경제 기여 + 기부 문화 확산 + 소외 지역 홍보**까지 이뤄낼 수 있는 플랫폼이 필요하다고 느꼈습니다.
